棒棒u!

含官方網站及「手機變魔術」書中網頁魔術

網站使用 Bootstrap Responsive Template，不管是電腦、平板還是手機可以自動適應畫面做調整， 加入 Google Analytics 做流量追蹤，DNS 則使用 AWS Route53，可以方便快速和 AWS 的其他服務結合。

官網部份靜態頁面放在 GitHub，除了免費空間和流量還可以使用 GitHub 的全球 DNS， 並且直接使用 GitHub 的版本控制，1 秒鐘即可切換網頁版本

動態程式像訂閱、發送 Email 等部份則是使用 AWS APIGateway，程式使用 Python 放在 AWS Lambda， 這兩個服務 AWS 會適需求自動擴充，佇列緩衝，不需去煩惱流量變化伺服器壓力增加的問題。

動態資料如訂閱者的 Email 將由 Lambda 寫入 AWS 的 NoSQL 服務 DynamoDB，可以做到幾乎即時的資料存取， 而且 AWS 內部服務的流量是不另外計費的。 電子郵件使用 Mailgun 做收信，自動轉寄到常用的 Gmail，使用 AWS SES 發信，設定 SMTP 到個人 Gmail，可以同介面快速切換發送個人信件或公司信件，SES 跟 Mailgun 也有提供發送電子報的 API，並且做使用者收信、開信、檢舉的各種追蹤。

「手機變魔術」書中約有一個網頁魔術，皆使用 JQuery 完成，無需任何動態資料，因此將檔案放在 AWS S3，搭配 Cloud Flare CDN 服務減少 S3 流量跟加速全球各地的讀取，尤其 Cloud Flare 在 2016 年建立台灣機房，台灣使用者速度上將會有更佳體驗。
